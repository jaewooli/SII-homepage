Curriculum

SII에서 엄선하여 진행하는 분야별 및 연차별 정밀 보안 학습 로드맵입니다.

[PHASE_01]

Web Hacking

웹 애플리케이션의 내부 구조적 아키텍처 및 OWASP Top 10, 권한 오용, 각종 인젝션 공격 등 핵심 취약점을 진단/분석합니다.

  • HTTP/HTTPS 프로토콜 원리
  • SQL Injection / XSS 심화
  • SSRF / File Inclusion 취약점
  • Web Exploit & RCE 시나리오
[PHASE_02]

System Hacking

컴퓨터 시스템 아키텍처, ELF 바이너리 로딩, OS 메모리 보호 기법을 우회하고 제어 흐름을 하이재킹하는 공격 기술을 익힙니다.

  • x86/x64 어셈블리어 분석
  • Stack & Heap Overflow
  • RTL & Return-Oriented Programming
  • ASLR / NX / Canary 우회 기법
[PHASE_03]

Reverse Engineering

컴파일된 기계어 바이너리를 역추적하여 소스 코드가 없는 프로그램의 흐름과 내부 정밀 로직을 분석 및 분석 무력화를 우회합니다.

  • Static / Dynamic 리버싱 기초
  • IDA Pro & Ghidra 디컴파일
  • 안티 디버깅 및 난독화 해제
  • 패킹 바이너리 언패킹 실습
[PHASE_04]

Cryptography

정보 은닉을 위한 수론 기반의 고전 암호부터 현대 표준 암호 알고리즘의 대칭/비대칭 설계 및 취약점 수학적 증명을 분석합니다.

  • Block Cipher & AES 대칭키
  • RSA / ECC 비대칭키 수학적 이해
  • Hash 충돌 및 MD5/SHA 공격
  • 현대 암호화 프로토콜 취약성
