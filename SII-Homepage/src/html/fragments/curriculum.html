Curriculum

SII에서 진행되는 주차별 및 분야별 세부 보안 학습 로드맵입니다.

Phase 1

Web Hacking

웹 애플리케이션의 동작 방식 이해 및 취약점(OWASP Top 10, Injection, XSS, CSRF 등) 분석 기법을 실습합니다.

  • HTTP 프로토콜 분석
  • SQL Injection / XSS
  • File Inclusion / Upload
  • Web exploit 실습
Phase 2

System Hacking

컴퓨터 아키텍처, 메모리 구조(Stack, Heap) 및 시스템 취약점(Buffer Overflow, RTL, ROP) exploit을 설계합니다.

  • 어셈블리어 기초
  • Buffer Overflow
  • RTL & ROP 기법
  • ASLR / DEP 우회
Phase 3

Reverse Engineering

바이너리 파일 분석, 디버깅 및 역공학 툴(IDA Pro, Ghidra, x64dbg) 사용법을 익히고 소프트웨어의 로직을 파악합니다.

  • Static/Dynamic 분석
  • IDA / Ghidra 분석
  • 안티 디버깅 기법
  • 패킹 / 언패킹
Phase 4

Cryptography

대칭키/비대칭키 암호화 알고리즘, 해시 함수 및 현대 암호 표준 프로토콜 분석을 다룹니다.

  • AES / DES 대칭키
  • RSA 공개키 알고리즘
  • 해시 함수 충돌 분석
  • 현대 암호 프로토콜
