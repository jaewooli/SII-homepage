Web Hacking

웹 구조적 취약점 분석, OWASP Top 10 및 RCE 취약점 심화 진단 과정입니다.

Week 01

Web Basics & Injection

HTTP 프로토콜 동작 원리를 이해하고, 입력값 검증 부재로 인해 발생하는 SQL 인젝션 공격 기법을 익힙니다.

  • HTTP Request/Response 구조와 쿠키/세션
  • SQL Injection 기본 및 Blind SQL Injection
  • XSS (Cross-Site Scripting) 취약점 유형
  • CSRF (Cross-Site Request Forgery) 동작 메커니즘
Week 02

Server Side Vulnerabilities

서버 상에서 파일 처리 및 외부 요청 시 발생하는 보안 취약점을 식별하고 익스플로잇하는 능력을 배웁니다.

  • File Upload 및 Webshell 업로드 공격
  • LFI (Local File Inclusion) / RFI 취약점
  • Command Injection 및 시스템 명령어 우회
  • SSRF (Server-Side Request Forgery) 개념 및 실습
Week 03

Advanced Web Exploit & RCE

최신 웹 애플리케이션의 복잡한 로직 버그와 원격 코드 실행(RCE)으로 연결되는 고급 취약점 시나리오를 심층 탐구합니다.

  • XML External Entity (XXE) 주입 취약점
  • PHP Object Injection & 역직렬화(Deserialization) 취약점
  • OAuth 2.0 / Webhook 로직 취약성 분석
  • 복합 취약점을 연쇄한 RCE 시나리오 설계
