Projects

Project 01

Wargame Solves Tracker (INHACK Extension)

Research 02

Visual Block Content Editor Engine

JSON 기반의 동적 템플릿 컴파일 기법을 활용하여 퍼블릭 페이지 요소를 데이터 코럽션 없이 안전하게 인라인 위지윅(WYSIWYG) 드래그 및 에디팅하는 포탈 코어 모듈입니다.
Tool 03
