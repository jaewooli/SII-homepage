⚠️ 공유 세션을 사용하는 동안에는 INHACK 확장 프로그램을 반드시 활성화해 주시고, 확장 프로그램이 비활성화된 상태에서 드림핵 사이트 로그아웃을 자제해 주세요. 비활성화 상태에서 로그아웃 시 다른 사용자의 세션 연동도 함께 해제됩니다.
공유 세션 연동
설치된 INHACK 크롬 확장 프로그램을 사용하여 공용 드림핵 세션을 브라우저에 동기화하고 연동합니다.
확장 프로그램 설치
INHACK 크롬 확장 프로그램이 없거나 비활성화되어 있는 경우, 확장 프로그램을 다운로드하여 브라우저에 수동으로 설치해 주세요.
확장 프로그램 다운로드 (.ZIP)
📦 설치 완료 후 이 페이지를 새로고침(F5)하면 자동으로 확장 프로그램이 인식됩니다.
